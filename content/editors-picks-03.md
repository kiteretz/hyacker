## 回答

セマンティック HTML とは、コンテンツの意味を表すタグを使って HTML を書くことです。`<div>` や `<span>` の代わりに `<header>`、`<nav>`、`<main>`、`<article>` などを使います。

```html
<!-- セマンティックでない例 -->
<div class="header">
  <div class="nav">...</div>
</div>

<!-- セマンティックな例 -->
<header>
  <nav>...</nav>
</header>
```

## 解説

### アクセシビリティの向上

スクリーンリーダーなどの支援技術がページ構造を正しく解釈できるようになります。視覚障がいのあるユーザーにとって、ナビゲーションの位置やメインコンテンツの場所が把握しやすくなります。

### SEO への効果

検索エンジンはセマンティックな HTML から各要素の重要度を正確に判断できます。`<h1>` タグの内容が最も重要な見出しとして扱われ、検索結果に影響します。

### 保守性の向上

コードの意味が明確になるため、チームでの開発や後からのコード修正がしやすくなります。

- `<header>` : ページやセクションのヘッダー
- `<nav>` : ナビゲーションリンク
- `<main>` : メインコンテンツ
- `<article>` : 独立したコンテンツ
- `<footer>` : ページやセクションのフッター
