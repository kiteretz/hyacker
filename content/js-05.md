## 回答

DOM 操作は `document.querySelector` で要素を取得し、プロパティやメソッドで変更します。

```js
// 要素の取得
const title = document.querySelector('h1');
const buttons = document.querySelectorAll('.button');

// テキスト変更
title.textContent = '新しいタイトル';

// HTML 変更（信頼できるコンテンツのみ）
title.innerHTML = '<em>強調</em>タイトル';

// クラスの操作
title.classList.add('highlight');
title.classList.remove('old-class');
title.classList.toggle('active');

// 属性の操作
const link = document.querySelector('a');
link.setAttribute('href', 'https://example.com');
link.getAttribute('href'); // 値を取得
```

## 解説

### 要素の作成と追加

```js
// 新しい要素を作成
const newItem = document.createElement('li');
newItem.textContent = '新しいアイテム';

// 末尾に追加
document.querySelector('ul').appendChild(newItem);

// 任意の位置に追加
const list = document.querySelector('ul');
list.insertAdjacentHTML('beforeend', '<li>末尾に追加</li>');
list.insertAdjacentHTML('afterbegin', '<li>先頭に追加</li>');
```

### 要素の削除

```js
const element = document.querySelector('.old');
element.remove(); // 要素を削除
```

### スタイルの変更

```js
const box = document.querySelector('.box');
box.style.backgroundColor = 'red';
box.style.fontSize = '24px';

// まとめてスタイルを適用するには classList を使う方が良い
box.classList.add('highlighted');
```
