## 回答

`localStorage` はブラウザを閉じても保持され、`sessionStorage` はタブを閉じると削除されます。

```js
// localStorage: 永続保存
localStorage.setItem('username', '田中');
const username = localStorage.getItem('username'); // '田中'
localStorage.removeItem('username');

// sessionStorage: セッション限り
sessionStorage.setItem('tempData', 'value');

// オブジェクトの保存（JSON に変換する必要がある）
const user = { name: '田中', age: 25 };
localStorage.setItem('user', JSON.stringify(user));
const storedUser = JSON.parse(localStorage.getItem('user'));
```

## 解説

### LocalStorage と SessionStorage の違い

| 項目 | LocalStorage | SessionStorage |
|------|-------------|----------------|
| 保持期間 | 明示的に削除するまで | タブを閉じるまで |
| スコープ | 同じオリジン全体 | 同じタブ内 |
| 容量 | 約 5MB | 約 5MB |

### すべてのデータをクリア

```js
localStorage.clear();
sessionStorage.clear();
```

### ユーティリティ関数でラップする

```js
const storage = {
  get(key, defaultValue = null) {
    try {
      const item = localStorage.getItem(key);
      return item ? JSON.parse(item) : defaultValue;
    } catch {
      return defaultValue;
    }
  },
  set(key, value) {
    localStorage.setItem(key, JSON.stringify(value));
  },
  remove(key) {
    localStorage.removeItem(key);
  }
};

storage.set('settings', { theme: 'dark', lang: 'ja' });
const settings = storage.get('settings', {});
```

### 注意点

- 文字列のみ保存できる（オブジェクトは JSON.stringify が必要）
- 機密情報（トークン、パスワード）は保存しない
- 容量オーバーは例外をスローする
