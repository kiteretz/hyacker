## 回答

クロージャとは、関数が定義されたスコープの変数を覚えている仕組みです。関数が外側のスコープにある変数を参照し続けます。

```js
function createCounter() {
  let count = 0; // この変数は counter 関数からアクセスできる

  return function counter() {
    count++;
    return count;
  };
}

const counter = createCounter();
console.log(counter()); // 1
console.log(counter()); // 2
console.log(counter()); // 3
```

## 解説

### カプセル化

クロージャを使うと、変数を外部から直接アクセスできないようにできます。

```js
function createBankAccount(initialBalance) {
  let balance = initialBalance; // プライベートな変数

  return {
    deposit(amount) {
      balance += amount;
      return balance;
    },
    withdraw(amount) {
      if (amount > balance) throw new Error('残高不足');
      balance -= amount;
      return balance;
    },
    getBalance() {
      return balance;
    }
  };
}

const account = createBankAccount(10000);
account.deposit(5000);  // 15000
account.withdraw(3000); // 12000
// balance に直接アクセスはできない
```

### ファクトリ関数

```js
function createMultiplier(multiplier) {
  return (number) => number * multiplier;
}

const double = createMultiplier(2);
const triple = createMultiplier(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```
