## 回答

分割代入でオブジェクトや配列から値を取り出し、スプレッド演算子でオブジェクト・配列を展開できます。

```js
// オブジェクトの分割代入
const user = { name: '田中', age: 25, role: 'admin' };
const { name, age } = user;
// name: '田中', age: 25

// デフォルト値と別名
const { name: userName = '名無し', role = 'user' } = user;

// 配列の分割代入
const [first, second, ...rest] = [1, 2, 3, 4, 5];
// first: 1, second: 2, rest: [3, 4, 5]

// スプレッド演算子でオブジェクトをコピー・マージ
const updatedUser = { ...user, age: 26 };
const merged = { ...user, ...{ role: 'editor', level: 2 } };
```

## 解説

### 関数の引数での分割代入

```js
// 関数の引数をオブジェクトで受け取る
function renderCard({ title, description, tags = [] }) {
  console.log(title, description, tags);
}

renderCard({
  title: '記事タイトル',
  description: '説明文',
  tags: ['HTML', 'CSS'],
});
```

### ネストしたオブジェクトの分割代入

```js
const data = {
  user: {
    name: '田中',
    address: {
      city: '東京'
    }
  }
};

const { user: { name, address: { city } } } = data;
```

### 配列のスプレッド

```js
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// 配列を結合
const combined = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

// 配列をコピー
const copy = [...arr1];
```
