## 回答

`map`、`filter`、`reduce` は配列を変換・フィルタリング・集計するための高階関数です。

```js
const numbers = [1, 2, 3, 4, 5];

// map: 各要素を変換した新しい配列を返す
const doubled = numbers.map(n => n * 2);
// [2, 4, 6, 8, 10]

// filter: 条件を満たす要素だけの新しい配列を返す
const evens = numbers.filter(n => n % 2 === 0);
// [2, 4]

// reduce: 配列を1つの値にまとめる
const sum = numbers.reduce((acc, n) => acc + n, 0);
// 15
```

## 解説

### map の活用例

```js
const users = [
  { name: '田中', age: 25 },
  { name: '佐藤', age: 30 },
];

// オブジェクトの配列から名前だけ取り出す
const names = users.map(user => user.name);
// ['田中', '佐藤']
```

### filter の活用例

```js
const products = [
  { name: 'A', price: 500, inStock: true },
  { name: 'B', price: 1200, inStock: false },
  { name: 'C', price: 800, inStock: true },
];

// 在庫ありで価格が1000円以下の商品
const available = products.filter(p => p.inStock && p.price <= 1000);
```

### reduce の活用例

```js
// オブジェクトの配列を集計
const total = products.reduce((sum, p) => sum + p.price, 0);

// 配列をオブジェクトに変換
const byName = products.reduce((obj, p) => {
  obj[p.name] = p;
  return obj;
}, {});
```

### チェーンして使う

```js
const result = numbers
  .filter(n => n > 2)    // [3, 4, 5]
  .map(n => n * 2)       // [6, 8, 10]
  .reduce((a, b) => a + b, 0); // 24
```
