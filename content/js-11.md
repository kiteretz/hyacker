## 回答

`console` オブジェクトのメソッドと、ブラウザの DevTools のブレークポイントを活用してデバッグします。

```js
// 基本的なログ出力
console.log('通常のログ');
console.info('情報');
console.warn('警告');
console.error('エラー');

// オブジェクトの中身を展開して表示
console.log({ name: '田中', age: 25 });

// テーブル形式で表示
console.table([
  { name: '田中', age: 25 },
  { name: '佐藤', age: 30 },
]);

// 処理時間の計測
console.time('処理時間');
// ... 計測したい処理 ...
console.timeEnd('処理時間');
```

## 解説

### console.group でログをまとめる

```js
console.group('ユーザー情報');
console.log('名前:', '田中');
console.log('年齢:', 25);
console.groupEnd();
```

### debugger ステートメント

コードに `debugger;` と書くと、DevTools が開いている場合にその行でブレークポイントが設定されます。

```js
function calculateTotal(items) {
  debugger; // ここで実行が停止する
  return items.reduce((sum, item) => sum + item.price, 0);
}
```

### DevTools でのデバッグ手順

1. F12 または Command+Option+I で DevTools を開く
2. Sources タブを開く
3. コードの行番号をクリックしてブレークポイントを設定
4. ページを操作してブレークポイントで停止させる
5. 右パネルで変数の値を確認する

### よくあるデバッグのポイント

- `typeof` で変数の型を確認する
- `JSON.stringify(obj, null, 2)` でオブジェクトを整形して出力
- 条件付きブレークポイントで特定条件のみ停止させる
