## 回答

アロー関数は `=>` を使って簡潔に関数を書ける構文です。

```js
// 通常の関数
function add(a, b) {
  return a + b;
}

// アロー関数
const add = (a, b) => a + b;

// 引数が1つの場合、括弧を省略可
const double = x => x * 2;

// 複数行の場合、波括弧と return が必要
const greet = (name) => {
  const message = `こんにちは、${name}！`;
  return message;
};

// オブジェクトを返す場合は括弧で囲む
const createUser = (name) => ({ name, createdAt: new Date() });
```

## 解説

### this の扱いの違い

アロー関数は自身の `this` を持たず、外側のスコープの `this` を参照します。

```js
class Timer {
  constructor() {
    this.seconds = 0;
  }

  start() {
    // アロー関数: this は Timer インスタンスを参照
    setInterval(() => {
      this.seconds++;
    }, 1000);

    // 通常の関数: this は undefined（strict mode）
    setInterval(function() {
      this.seconds++; // エラー
    }, 1000);
  }
}
```

### アロー関数が使えない場面

- オブジェクトのメソッドとして `this` を使う場合
- `arguments` オブジェクトが必要な場合
- コンストラクタ関数（`new` キーワードで呼び出す）として使う場合
